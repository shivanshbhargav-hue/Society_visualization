# Dashboard for Society Level for Dark Stores

A React + TypeScript dashboard application built with Vite, Tailwind CSS, and shadcn/ui.

## Getting Started

### Prerequisites

- Node.js (v18+)
- npm

### Installation

```sh
npm install
```

### Development

```sh
npm run dev
```

### Build

```sh
npm run build
```

### Testing

```sh
npm test
```
